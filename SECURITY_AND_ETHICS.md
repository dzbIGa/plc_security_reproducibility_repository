# Security and Ethics Statement

This repository was prepared to support scientific reproducibility while minimizing operational risk.

It intentionally provides:
- defensive configuration examples,
- anonymized and synthetic data,
- transparent analysis code,
- metric calculation and statistical validation scripts.

It intentionally omits:
- exploit automation,
- malicious payloads,
- sensitive device dumps,
- credentials,
- raw real-world network captures that could aid misuse.

The package is suitable for peer review, archival, and public release as a reproducibility supplement.
